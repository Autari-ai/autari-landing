# Template Study — EXHAUSTIVE reference (single source of truth)

Every one of the 15 reference `.webm` templates, studied frame-by-frame (Playwright contact sheets +
full-size frames) AND in motion (ffmpeg GIFs). This doc is written to be **self-sufficient**: assume the
source videos are gone and this is the only record. For each template: brand, layout zone-by-zone,
palette, typography, and EVERY motion (entrance / ambient / transition / scroll). Then the synthesized
Autari system and the reusable technique library.

Target build: **Autari** — dark-cinematic + green-glow landing page, CTA = **Book a call → noor@autari.co.uk**,
positioning broadened to "AI employees that run repetitive work across ANY field" (not just engineering).

Source pattern: `https://figma-templates-en.vercel.app/assets/<template>/<name>.webm`
Local copies: `scratchpad/webm/*.webm`, motion GIFs `scratchpad/gifs/*.gif`, frames at repo root `t##-*.jpeg`.

## Reusable technique library (the vocabulary, deduped)

1. **Animated cinematic hero subject** — a 3D/rendered focal object that rotates / blooms / flows / splashes / dissolves. *(T1,T6,T8,T9,T21,T22,T24,T28)*
2. **Glass content panel, 3D perspective tilt-in** — the hero/section panel animates from an angled tilt to flat, or scales in. *(T21,T22,T28,T37)*
3. **Ambient color-bleed backdrop** — the media's dominant colors blurred to fill the page behind its frame; shifts as content changes. *(T9,T27,T37,T38)*
4. **Monochrome brand-tint on ALL media** — one hue graded across every asset for a single-film feel. *(T26)* → we tint green/teal.
5. **Text-behind-subject** — the big headline sits behind the 3D hero object, partially occluded. *(T1,T9,T24)*
6. **Floating glass stat chips** — big number + tiny unit/superscript + small label; frosted, thin border. *(T22,T28,T29,T38)*
7. **Numbered index markers** — `[001][002][003]` or "01 —— 05" for features / steps / slides. *(T8,T22,T24,T27,T29,T37,T38)*
8. **Scroll scene-morph / carousel** — sections dissolve into each other, or a horizontal slider with peeking neighbors. *(T1,T6,T27)*
9. **Living micro-motion** — particles drift, glows pulse, cards bob/rotate; the page is never fully still. *(OV,T1,T38)*
10. **Circular badges** — "SCROLL DOWN", "Watch Demo ▶", radial-dotted "motion loop" seal, rotating "PURE X POWER" seal. *(T6,T21,T24,T28,T37,OV)*
11. **Minimal pill nav** — centered or split; utility icons (search/cart/avatar/grid); one action pill. *(T6,T8,T21,T28,T29,T37,T38)*
12. **Dual CTA** — filled pill + plain text link beside it. *(T37)* → "Book a call" + "See how it works".
13. **Product-in-context device mockup** — laptop/iMac in a real environment showing the site. *(T2,T6)* (optional for a "watch it work" section).
14. **Two-tone / mixed-weight / italic-emphasis headlines** — emphasize one word by weight, italic, or color. *(T6,T22,T37)*
15. **Vertical social / thumbnail rail** — thin column of icons or thumbnails at a screen edge. *(T6,T24,T37)*
